# SPEC 인터뷰

## Description

SPEC.md 파일을 기반으로 심층 인터뷰를 진행하여 완성도 높은 스펙 문서를 작성하는 프롬프트

## Tags

spec, interview, requirements, documentation

## Usage

1. 프로젝트 루트에 `SPEC.md` 파일 생성 (초안 작성)
2. 이 프롬프트 실행
3. AI의 질문에 답변하며 스펙 구체화
4. 인터뷰 완료 후 최종 스펙 파일 생성

## Prompt

```
@SPEC.md파일을 읽고 AskUserQuestionTool을 사용하여 기술적 구현, UI & UX, 우려 사항, 트레이드오프 등 모든 측면에 대해 저를 상세히 인터뷰해 주세요. 질문은 뻔하거나 상투적이지 않아야 하며, 매우 심층적으로 접근하여 내용이 완성될 때까지 인터뷰를 계속 이어가야 합니다. 인터뷰가 끝나면 스펙을 파일에 작성하세요.
```
