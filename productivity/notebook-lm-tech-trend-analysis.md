# Notebook LM 기술 동향 분석 프롬프트

## Description

Notebook LM에서 기술 소스를 심층 분석할 때 사용하는 프롬프트 가이드

## Tags

notebook-lm, ai, tech-trends, analysis, research

## Usage

1. Contents Hub 다이제스트 이메일에서 "URL 복사하기" 클릭
2. Notebook LM에 URL 목록 붙여넣기
3. 아래 프롬프트 중 목적에 맞는 것 선택하여 사용

## Prompt

### 심층 분석 (Deep Dive)

```
오늘 소스들에서 발견되는 AI/기술 트렌드의 공통 패턴을 분석해줘.
- 여러 소스에서 반복적으로 언급되는 주제는?
- 서로 상충되는 관점이 있다면?
- 실무에 바로 적용할 수 있는 인사이트 3가지
- 추가로 탐색할 만한 관련 주제 제안
```

### 요약 (Summary)

```
각 소스의 핵심 메시지를 3문장 이내로 요약하고,
전체를 관통하는 하나의 큰 그림(메타 인사이트)을 도출해줘.
```

### 토론 (Debate)

```
오늘 소스들 중 가장 논쟁적인 주장이나 예측을 찾아서,
찬반 양측의 근거를 정리하고 균형 잡힌 시각을 제시해줘.
```

### 짧은 버전 (Quick)

```
AI 트렌드 + 실무 적용점 + 소스 간 연결고리 분석해줘
```

## Tips

- 심층 분석은 10분 이상 소요, 출퇴근길에는 요약/짧은 버전 추천
- 토론 모드는 논쟁적인 주제(AGI 타임라인, 오픈소스 vs 클로즈드 등)에 효과적
- 오디오 개요 기능과 함께 사용하면 팟캐스트처럼 들을 수 있음
