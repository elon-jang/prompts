# Excel 이슈 데이터 피벗 및 시트 분리

## Description

엑셀 이슈 목록을 피벗 테이블로 요약하고 조합별 시트로 분리

## Tags



## Usage



## Prompt

```
## 기능 1: 데이터 분석 및 시트 분리
입력된 이슈 목록 엑셀 파일을 읽어 다음 작업을 수행하세요.
1. **피벗 테이블 생성:** - 행: `현장명`, `장소_분류`
   - 열: `심각도`
   - 값: 개수(Count)
   - 결과는 새로운 시트에 저장하세요.
2. **상세 시트 분리:**
   - 전체 데이터를 `{{현장명}}_{{장소_분류}}` 조합별로 필터링하세요.
   - 각 데이터는 `심각도` 기준 오름차순으로 정렬하세요.
   - 필터링 된 각 데이터를 `{{현장명}}_{{장소_분류}}`라는 이름의 개별 시트로 저장하세요.
```
