# 다중 소스 분석 보고서 생성기

## Description

여러 소스의 공통점과 독창적 주장을 초등학생도 이해할 수 있게 정리

## Tags



## Usage



## Prompt

```
첨부된 소스 속 내용을 100% 이해할 수 있는 보고서를 만들어 줘.
또한 해당분야의 일반적인 내용과 다른, 독창적인 주장 이나 지식을 따로 구분해서 정리해 줘.
추가로 이런 주장이나 지식에 대한 구체적인 근거 자료도 함께 찾아서 정리해 줘.
보고서는 초등학생도 이해하기 쉽게 작성해 줘.

첨부된 소스의 내용 중 공통적인 부분을 정리하고, 소스별로 독창적인 내용을 근거와 함께 정리해 줘.
```
