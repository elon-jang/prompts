# 공공 데이터 포털 크롤링 (bizinfo.go.kr)

## Description

기업지원 공고 크롤링 및 마감일 필터링 스크립트

## Tags

python, crawling, beautifulsoup, selenium, excel

## Usage



## Prompt

```
# 역할
당신은 파이썬 웹 크롤링 전문가입니다. `requests`와 `BeautifulSoup` (필요 시 `selenium`)을 사용하여 공공 데이터 포털의 정보를 수집하는 스크립트를 작성해 주세요.

# 타겟 URL
- 기본 URL: `https://www.bizinfo.go.kr/web/lay1/bbs/S1T122C128/AS/74/list.do`
- 파라미터: `rows=15`, `cpage=1` (페이지 번호 1~20까지 순회)

# 수집 데이터
각 공고 목록에 대해 다음 정보를 수집하여 엑셀로 저장하세요.
1. 목록 표에 있는 모든 컬럼 정보.
2. 상세 페이지 URL (링크).
3. 상세 페이지 내부의 '사업개요' 텍스트 전체.

# 데이터 가공 및 필터링 로직 (매우 중요)
수집 과정에서 다음 로직을 반드시 적용하세요.

1. **날짜 분리:** - '신청기간' 컬럼의 텍스트를 파싱하여 `신청 시작일`과 `신청 종료일` 두 개의 열로 나누어 저장하세요.
   - 예외 처리: 신청기간이 '예산 소진시까지', '상시 접수', '추후 공지', '세부사업별 상이'인 경우 날짜를 빈칸으로 두거나 원본 텍스트를 비고란에 남기세요.

2. **조건부 필터링 (오늘 날짜 기준):**
   - 오늘 날짜가 `신청 종료일`보다 늦은 경우(이미 마감된 공고)는 결과에서 제외하세요.
   - **단, 예외 텍스트('예산 소진시까지', '상시 접수', '추후 공지', '세부사업별 상이')가 포함된 공고는 마감일이 정해지지 않은 것으로 간주하여 무조건 포함시키세요.**

# 출력물
- 모든 결과가 담긴 `crawling_result.xlsx` 파일.
```
