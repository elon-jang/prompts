# Excel 파일 병합 (Merge)

## Description

여러 엑셀 파일을 하나로 병합하고 서식 설정

## Tags

excel, merge, automation

## Usage



## Prompt

```
## 기능 2: 파일 병합 (Merge)
스크립트가 실행되는 폴더 내의 모든 엑셀 파일을 하나로 취합하여 `Merged_Result.xlsx`로 저장하세요.
1. **헤더 처리:** 첫 번째 파일의 헤더(1행)만 유지하고, 나머지 파일들은 2행부터 데이터만 가져와 합치세요.
2. **서식 유지:**
   - 모든 셀의 '텍스트 줄바꿈(Wrap Text)' 속성을 활성화하세요.
   - A열(1열)의 너비는 10, 나머지 모든 열의 너비는 25로 설정하세요.
```
