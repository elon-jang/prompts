# Word 문서 대량 생성 (Mail Merge)

## Description

엑셀 데이터로 워드 양식을 치환하여 대량 문서 생성

## Tags

python, word, docx, automation, mail-merge

## Usage



## Prompt

```
# 역할
당신은 파이썬 사무 자동화 전문가입니다. `python-docx`와 `pandas`를 사용하여 워드 문서를 대량으로 생성하는 스크립트를 작성해 주세요.

# 입력 파일
1. 데이터 파일: 엑셀 파일 (직원 정보 리스트)
2. 양식 파일: `비밀유지서약서 양식.docx` (내부에 `{{소속팀}}`, `{{이름}}`, `{{생년월일}}`, `{{입사일}}` 텍스트가 포함됨)

# 기능 요구사항
엑셀 파일의 각 행(Row)을 순회하며 다음 작업을 수행하세요.

1. **데이터 치환:** - 양식 파일 내의 `{{소속팀}}`, `{{이름}}`, `{{생년월일}}`, `{{입사일}}` 부분을 엑셀의 해당 열 데이터로 정확히 찾아 교체하세요. (Paragraph와 Table 내의 텍스트 모두 처리 필요)
2. **폴더 생성:** - 스크립트 실행 '오늘 날짜'로 된 폴더를 생성하세요. (예: `20251102`)
3. **파일 저장:**
   - 치환이 완료된 문서를 위에서 생성한 폴더 안에 `비밀유지서약서_{{이름}}.docx` 형식으로 저장하세요.

# 주의 사항
- 워드 문서의 기존 서식(폰트, 크기, 정렬 등)이 깨지지 않도록 `python-docx`의 `runs` 객체를 활용하여 텍스트만 교체하는 로직을 구현하세요.
```
