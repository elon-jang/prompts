# 인체공학 자세 가이드 이미지 프롬프트

## Description

사무실 인체공학 자세를 기술 도표와 함께 보여주는 이미지 생성 프롬프트

## Tags

image, ergonomic, office, posture, educational

## Usage

Midjourney, DALL-E 등 AI 이미지 생성 도구에 프롬프트로 입력

## Prompt

```
밝고 현대적인 사무실을 배경으로, 기술적 도표 요소가 겹쳐진(오버레이 된) 사실적인 측면 사진입니다. 스마트 캐주얼 차림의 전문직 종사자가 깨끗한 책상 앞 고급 인체공학 의자에 완벽한 자세로 앉아 눈높이에 위치한 모니터를 바라보고 있습니다. 등은 곧게 펴져 있고 엉덩이는 무릎보다 약간 높습니다.

은은하게 빛나는 흰색 건축용 라인과 각도 표시가 이미지 위에 그려져 있습니다.

눈에서 모니터로 향하는 선: "시선: 0°~30°"

팔꿈치의 곡선: "팔꿈치: 90°~110°"

무릎의 곡선: "무릎: 90°~100°"

전체적인 스타일은 세련되고 깔끔하며 교육적입니다. 자연광이 실내를 가득 채우고 있으며, 8K 고해상도 화질입니다.
```
