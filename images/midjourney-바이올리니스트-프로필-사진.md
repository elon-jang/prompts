# Midjourney 바이올리니스트 프로필 사진

## Description

증명사진의 얼굴을 전문 바이올리니스트 포즈와 합성하여 프로필 사진을 생성하는 Midjourney 프롬프트

## Tags

midjourney, image-generation, portrait, face-swap, violinist

## Usage

1. Midjourney 웹사이트 또는 Discord에서 `/imagine` 클릭
2. 이미지 첨부 버튼으로 증명사진, 바이올린 참조사진 순서로 첨부
3. 아래 프롬프트 텍스트 입력 후 실행

## Prompt

```
전문 바이올리니스트의 프로필 사진. 첫 번째 이미지(증명사진)의 얼굴 특징과 표정을 극도로 사실적으로 유지하며, 두 번째 이미지의 포즈, 구도, 조명, 배경 분위기에 자연스럽게 합성. 고해상도 인물 사진, 시네마틱 조명, 얕은 심도 --ar 2:3 --v 6.0 --style raw
```

## Parameters

| 파라미터 | 값 | 설명 |
|---------|-----|------|
| `--ar` | 2:3 | 세로 비율 (인물 사진에 적합) |
| `--v` | 6.0 | Midjourney 버전 6 |
| `--style` | raw | 덜 가공된 자연스러운 스타일 |
