# 사무실 책상 아래 다리 스트레칭 영상 프롬프트

## Description

사무실 책상 아래에서 하는 다리 스트레칭 운동 시네마틱 영상 생성 프롬프트

## Tags

video, veo, office-exercise, cinematic

## Usage

AI 영상 생성 도구에 프롬프트로 입력

## Prompt

```
현대적인 미니멀리즘 사무실 책상 아래 공간을 낮은 카메라 각도에서 촬영한 고화질 시네마틱 영상. 정장 바지와 깨끗한 운동화를 착용한 사람이 의자에 앉아 있다. 두 다리는 마주 보지 않고 나란하게 위치한다. 한쪽 다리를 책상 아래에서 앞쪽으로 곧게 펴서 아주 높이 들어 올리고, 발끝을 몸쪽으로 당긴 채 10초 동안 정지하며 버틴다. 그다지 티 나지 않게 통제된 움직임으로 다리를 천천히 내린 후, 반대쪽 다리도 똑같이 높이 들어 올려 10초간 유지한다. 바닥에는 밝은 자연광이 비치며, 전체적으로 깨끗하고 세련되며 운동을 권장하는 분위기다. 4K 화질, 슬로우 모션.
```
